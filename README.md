# 📝 議事録タイピング練習

会議の議事録作成スキルを向上させるためのタイピング練習Webアプリです。  
ブラウザの音声合成機能で読み上げられる日本語を聞き取り、キーボードで正確に入力する練習ができます。

## 🎮 遊び方

1. **難易度を選択** — 初級・中級・上級の3段階から選べます
2. **音声を再生** — 🔊ボタンで音声を再生します（各問題につき最大3回まで）
3. **入力** — 聞こえた内容をテキストエリアに入力します
4. **採点** — 「採点する」ボタンで得点が表示されます
5. **結果確認** — 正解との差分がハイライト表示されます

## 📊 採点基準

| 項目 | 配点 | 説明 |
|------|------|------|
| 正確さ | 0〜100点 | 正解テキストとの文字一致率 |
| 時間ボーナス | 0〜20点 | 制限時間内の完了速度に応じた加点 |

### 時間ボーナス詳細

- 制限時間の50%以内に完了: **+20点**
- 制限時間の75%以内に完了: **+10点**
- 制限時間内に完了: **+5点**
- 制限時間超過: **+0点**

## 🎯 難易度

| レベル | 内容 | 読み上げ速度 | 制限時間 |
|--------|------|-------------|---------|
| 🟢 初級 | 短い定型フレーズ | ゆっくり | 60秒 |
| 🟡 中級 | 業務文・議題説明 | 普通 | 90秒 |
| 🔴 上級 | 長文・専門用語 | 速い | 120秒 |

## 🌐 GitHub Pages で公開

このアプリは GitHub Pages でホスティングされています。
- https://yuzuafro.github.io/meeting-typing/

### 自分でデプロイする場合

1. このリポジトリをフォークまたはクローン
2. GitHub のリポジトリ設定で **Pages** > **Source** を **GitHub Actions** に設定
3. `main` ブランチにプッシュすると自動的にデプロイされます

## 🛠 技術スタック

- HTML / CSS / JavaScript（フレームワーク不使用）
- Web Speech Synthesis API（ブラウザ内蔵音声合成）
- GitHub Actions + GitHub Pages（デプロイ）

## ⚠ 注意事項

- Web Speech Synthesis API の対応状況はブラウザによって異なります
- 推奨ブラウザ: Google Chrome / Microsoft Edge
- 日本語音声が利用可能なブラウザで使用してください
