# 📝 議事録タイピング練習

会議の議事録作成スキルを向上させるためのタイピング練習Webアプリです。  
**音声モード**と**文字モード**の2種類の練習方法で、実践的な入力スキルを鍛えられます。

## 🎮 練習モード

### 🔊 音声モード
ブラウザの音声合成機能で読み上げられる日本語を聞き取り、キーボードで入力します。  
議事録の「耳で聞いて打ち込む」実務に近い練習ができます。

1. **モードタブ**で「🔊 音声モード」を選択
2. **難易度**を選択
3. **音声を再生**（各問題につき最大3回まで再生可能）
4. **聞こえた内容を入力**（初回再生終了後にタイマー開始）
5. **採点** → 正解との差分をハイライト表示で確認

### 📖 文字モード
画面に表示されたテキストを素早く正確に入力します。  
タイピング速度と正確さを鍛える練習ができます。

1. **モードタブ**で「📖 文字モード」を選択
2. **難易度**を選択
3. **テキストが表示**され、即タイマー開始
4. **文章を入力**（テキストのコピーは無効）
5. **採点** → 正解との差分をハイライト表示で確認

## 📊 採点基準

| 項目 | 配点 | 説明 |
|------|------|------|
| 正確さ | 0〜100点 | 正解テキストとの文字一致率（句読点は採点対象外） |
| 時間ボーナス | 0〜20点 | 制限時間内の完了速度に応じた加点 |

### 時間ボーナス詳細

| 完了タイミング | ボーナス |
|--------------|---------|
| 制限時間の50%以内 | **+20点** |
| 制限時間の75%以内 | **+10点** |
| 制限時間内 | **+5点** |
| 制限時間超過 | **+0点** |

> 句読点（、。！？など）の有無は正確さに影響しません。

## 🎯 難易度

### 🔊 音声モード

| レベル | 内容 | 読み上げ速度 | 制限時間 |
|--------|------|-------------|---------|
| 🟢 初級 | 短い定型フレーズ | ゆっくり | 60秒 |
| 🟡 中級 | 業務文・議題説明 | 普通 | 90秒 |
| 🔴 上級 | 長文・専門用語 | 速い | 120秒 |

### 📖 文字モード

| レベル | 内容 | 制限時間 |
|--------|------|---------|
| 🟢 初級 | 短い定型フレーズ | 40秒 |
| 🟡 中級 | 業務文・議題説明 | 60秒 |
| 🔴 上級 | 長文・専門用語 | 90秒 |

各難易度10問の問題プールから、毎回5問をランダムに出題します。

## ✨ その他の機能

- **🌙 ダークモード対応** — ヘッダーのボタンでライト/ダークを切り替え。システム設定にも自動追従
- **差分ハイライト** — 採点後、正解と入力の差分を文字単位でカラー表示
- **結果サマリー** — 全5問終了後に総合スコア・平均正確さ・最高スコアを表示

## 🌐 GitHub Pages で公開

このアプリは GitHub Pages でホスティングされています。

### 自分でデプロイする場合

1. このリポジトリをフォークまたはクローン
2. GitHub のリポジトリ設定で **Settings > Pages > Source** を **GitHub Actions** に設定
3. `main` ブランチにプッシュすると自動的にデプロイされます

## 🛠 技術スタック

- HTML / CSS / JavaScript（フレームワーク不使用）
- Web Speech Synthesis API（ブラウザ内蔵音声合成）
- GitHub Actions + GitHub Pages（CI/CDデプロイ）

## ⚠ 注意事項

- Web Speech Synthesis API の対応状況はブラウザによって異なります
- 推奨ブラウザ: Google Chrome / Microsoft Edge
- 日本語音声が利用可能なブラウザで使用してください（音声モード利用時）
